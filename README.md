# Lab 2

## Overview
This lab provided an opportunity to apply fundamental machine learning concepts to real-world datasets using Python and scikit-learn. Through supervised learning (regression and classification) and unsupervised learning (K-Means clustering), I explored the complete machine learning workflow. From data exploration and preprocessing to feature engineering, model training, evaluation, and the identification of overfitting. Working through these tasks deepened my understanding of how different machine learning techniques are applied and the importance of building models that generalise well to unseen data.
